Any page of a website -- WeLive Online Customer Service Loading Demo

WeLive can be called by any website(Cross Domain), even the localhost website page, all you need to do is inserting below code into the <html> tag of a page:
<script type="text/javascript" charset="UTF-8" src="http://your_domain/dir_welive_installed/welive.js"></script>

Mobile devices such as mobile phone and pad can also be adapted, please browse this page on the mobile device for testing.
